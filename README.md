Website mobile-first para descobrir, criar e compartilhar receitas. Com passo a passo ilustrado, sistema de listas de compras e construído com tecnologias modernas para oferecer a melhor experiência.

Em desenvolvimento! (Work in progress!)

Stack:
 - Framework: Next.js
 - Linguagem: Typescript
 - Banco de dados: Postgres Neon
 - ORM: Prisma
 - Autenticação: BetterAuth
 - Upload de imagens: Cloudinary
 - Animações: Framer Motion
 - Estilos: Tailwind CSS + Shadcn/UI
 - Deploy: Vercel
